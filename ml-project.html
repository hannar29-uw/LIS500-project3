Project Scope

The scope of our LIS 500 Project 3 centers on exploring how machine learning models classify and interpret human skin tones and visualizing user interactions with interactive code using p5.js. Our main goal was to build a simple Teachable Machine model capable of receiving webcam input and attempting to categorize skin tones in real time, while documenting biases and limitations that arise due to training data constraints.

Additionally, we created a p5.js project to demonstrate interactive programming and dynamic visualizations. This portion of the project allowed us to explore creative coding, user input, and data-driven visuals, complementing our machine learning component.

In Scope

  • Building a Teachable Machine model to classify skin tones using webcam input
  • Testing and documenting model behavior and biases
  • Creating interactive visualizations using p5.js
  • Producing videos and a webpage to demonstrate both projects
  • Collaborating as a team to establish a consistent style guide and workflow

Out of Scope

  • Advanced machine learning techniques beyond the Teachable Machine platform
  • Creating a fully deployed web application with server-side code
  • Collecting a fully representative dataset for skin tones
  • Integrating the p5.js project directly with the Teachable Machine model

Overall, the project scope allowed us to explore machine learning and creative coding in a controlled environment, focusing on understanding model behavior, user interaction, and bias in algorithms while producing tangible outputs for demonstration.
